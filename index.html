História dos computadores

A história dos computadores começa muito antes de termos laptops e smartphones. No século XIX, Charles Babbage projetou a "Máquina Analítica", considerada o primeiro modelo de computador programável. Mas foi no século XX que os computadores começaram a evoluir de verdade, com a criação do ENIAC, o primeiro computador eletrônico geral, em 1945. Com o passar do tempo, os computadores ficaram menores, mais potentes e acessíveis, abrindo caminho para a internet e, consequentemente, para as tecnologias web que usamos hoje.

O que são Clients?

Quando falamos em desenvolvimento web, o termo "client" se refere ao dispositivo ou navegador que um usuário utiliza para acessar um site. Imagine que você está usando seu navegador (como o Chrome ou o Firefox) para abrir uma página da web. Nesse caso, seu navegador é o "client". Ele faz requisições para acessar informações hospedadas em um servidor e exibe essas informações para você.

O que são Servers?

Por outro lado, o "server"(servidor) é o local onde as informações e os recursos de um site estão armazenados. Ele é responsável por receber as requisições feitas pelos clients e enviar as respostas. Em termos simples, o servidor funciona como um entregador de informações, garantindo que os dados de um site cheguem até o navegador do usuário.

Como é uma estrutura básica de um HTML?

O HTML é a linguagem base para criar páginas da web. Ele estrutura o conteúdo que será exibido no navegador. Uma estrutura básica de HTML é bastante simples e tem a seguinte aparência:

        <!DOCTYPE html>
        <html>
        <head>
            <title>Minha Primeira Página</title>
        </head>
        <body>
            <h1>Bem-vindo ao meu site!</h1>
            <p>Este é um parágrafo de exemplo.</p>
        </body>
        </html>

Aqui está o que cada parte faz:

  • <!DOCTYPE html>: Declara que estamos usando HTML5.
  • <html>: Define o início e o fim do documento HTML.
  • <head>: Contém informações sobre a página, como o título.
  • <body>: É onde colocamos o conteúdo visível ao usuário, como textos, imagens e links.

Entendendo a diferença entre listas ordenadas e não ordenadas

No HTML, podemos usar listas para organizar informações. Existem dois tipos principais:

  1. Listas Ordenadas (<ol>):
    • São usadas quando a ordem dos itens é importante.
    • Os itens aparecem numerados.
    • Exemplo:
      •                 <ol>
                    <li>Primeiro item</li>
                    <li>Segundo item</li>
                    <li>Terceiro item</li>
                </ol>
  2. Listas Não Ordenadas (<ul>):
    • São usadas quando a ordem dos itens não importa.
    • Os itens aparecem com marcadores (bolinhas, quadrados, etc.).
    • Exemplo:
      •                 <ul>
                    <li>Primeiro item</li>
                    <li>Segundo item</li>
                    <li>Terceiro item</li>
                </ul>
